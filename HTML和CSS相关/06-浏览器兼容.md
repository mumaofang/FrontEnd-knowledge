# CSS兼容性问题及解决方案
    1. 默认内外边距差异 * { margin: 0; padding: 0; }
    2. 浮动布局导致Margin异常（IE6） .float-box { float: left; display: inline; }
    3. 小高度标签溢出（IE6/7） .small-height { height: 5px; overflow: hidden; line-height: 5px; }
    4. 图片默认间距 
        多个<img>标签并排时出现间隙。 为图片添加浮动：img { float: left; }
    5. min-height不兼容
        旧浏览器（如IE6）不支持min-height属性。 通过组合属性模拟最小高度：
        ```css
            .container { 
                min-height: 200px; 
                height: auto !important; 
                height: 200px; 
                overflow: visible;
            }
        ```
    6. 透明度兼容。 IE不支持opacity属性。 结合filter和opacity：
        ```css
            .transparent { 
                opacity: 0.8;         /* 现代浏览器 */
                filter: alpha(opacity=80);  /* IE */
            }
        ```
# JavaScript兼容性问题及解决方案
    1. 监听事件差异
        IE8及以下使用attachEvent，其他浏览器使用addEventListener  
        封装兼容性函数：  
        ```javascript
        function addEvent(element, type, handler) {
            if (element.addEventListener) {
                element.addEventListener(type, handler);
            } else if (element.attachEvent) {
                element.attachEvent('on' + type, handler);
            }
            }
        ```
    2. 获取元素属性
        IE允许直接通过element.value获取自定义属性，而其他浏览器需用getAttribute。  
        统一使用getAttribute： const value = element.getAttribute('data-custom');
